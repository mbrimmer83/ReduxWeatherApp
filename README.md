# ReduxWeatherApp

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?
Checkout this repo, install dependencies, then start the gulp process with the following:

```
> git clone https://github.com/mbrimmer83/ReduxWeatherApp.git
> cd ReduxWeatherApp
> npm install
> npm start
```

#### Getting started
```
> npm install
> npm start
```
